# Favicon 預覽工具

上傳一張圖，立即預覽它當作網站 favicon 在瀏覽器分頁上的樣子。

- 上傳（拖放或點選）後即時套用到分頁圖示，並在 16/32/64px × 深色/淺色分頁底下對照
- 內建防呆：格式、檔案大小、尺寸（非正方形/過小/過大）檢查與提示
- 支援深色 / 淺色模式切換（記住選擇）
- 純前端、零後端：圖片只在你的瀏覽器處理，不會上傳到任何伺服器

## 使用

直接開啟 `index.html` 即可；或部署到任何靜態主機（如 GitHub Pages）。

## 結構

```
index.html        主程式（HTML/CSS/JS 單檔）
assets/favicon.ico 預設分頁圖示
```
